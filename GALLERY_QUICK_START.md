# Gallery Management - Quick Start Guide

## üéØ Overview
The Gallery Management system allows you to manage all visual content displayed on your website's gallery page, including photos, videos, Instagram posts, and Instagram reels.

---

## üìç How to Access

### Method 1: From Sidebar
1. Login to admin dashboard at `/admin/login`
2. Click **"Gallery"** in the left sidebar navigation

### Method 2: From Dashboard
1. Go to Admin Dashboard at `/admin/dashboard`
2. Scroll to "Quick Management" section
3. Click the **"Gallery"** card

---

## ‚ûï Adding New Gallery Items

### Step-by-Step:
1. Click the **"Add New Item"** button (top right)
2. Select content type:
   - **Image** - Regular photos
   - **Video** - Video content
   - **Instagram Post** - Instagram image posts
   - **Instagram Reel** - Instagram video reels

3. Fill in the form:
   - **Title**: Name of the item (e.g., "Bridal Lehenga Collection")
   - **Category**: Select from dropdown (Sarees, Lehengas, Anarkalis, Suits, Kurtis)
   - **URL**: Direct link to the image or video
   - **Thumbnail** (for videos only): Preview image URL
   - **Instagram URL** (for Instagram content): Link to the Instagram post/reel
   - **Status**: Choose Active, Draft, or Archived

4. Click **"Add Item"**

### Example URLs:
- **Image**: `https://images.unsplash.com/photo-1610030469983-98e550d6193c?w=800&q=80`
- **Video**: `https://player.vimeo.com/external/370331493.sd.mp4`
- **Instagram**: `https://instagram.com/p/example123`

## ‚úèÔ∏è Editing Gallery Items

### Grid View:
1. Hover over any gallery item
2. Click the **"Edit"** button that appears
3. Modify the details in the modal
4. Click **"Update Item"**

### List View:
1. Find the item in the list
2. Click the **Edit icon** (pencil) on the right
3. Update the form
4. Click **"Update Item"**

---

## üóëÔ∏è Deleting Gallery Items

### Single Delete:
1. Hover over the item (Grid View) or find it (List View)
2. Click the **trash icon**
3. Item is deleted immediately

### Bulk Delete:
1. Select multiple items using checkboxes
2. Click **"Delete Selected"** button
3. All selected items are removed

## üì¶ Bulk Operations

### Selecting Items:
- Click checkboxes on individual items
- Or click **"Select All"** to select all visible items

### Bulk Actions:
1. **Set Active**: Make all selected items visible on website
2. **Set Draft**: Hide all selected items from website

- Click **"Deselect All"** button

## üé® Content Type Guide

### Image
- **Use for**: Product photos, lifestyle shots, behind-the-scenes
- **Required**: URL, Title, Category
- **Optional**: Instagram URL (if from Instagram)

### Video
- **Use for**: Product videos, tutorials, showcases
- **Required**: Video URL, Title, Category
- **Optional**: Thumbnail URL (recommended)

### Instagram Post
- **Use for**: Embedding Instagram image posts
- **Required**: Image URL, Title, Category
- **Optional**: Instagram URL (link to original post)

### Instagram Reel
- **Use for**: Embedding Instagram video reels
- **Required**: Video URL, Title, Category, Thumbnail
- **Optional**: Instagram URL (link to original reel)

## üí° Best Practices

### Image URLs:
- Use high-quality images (minimum 800px width)
- Ensure URLs are publicly accessible
- Use HTTPS URLs for security
- Consider using CDN for better performance

### Titles:
- Be descriptive and specific
- Include product name or theme
- Keep under 50 characters
- Use title case

### Categories:
- Choose the most relevant category
- Be consistent across similar items
- Use categories that match your products

### Status:
- Set to Draft while preparing content
- Change to Active when ready to publish
- Use Archived for seasonal or outdated content

### Organization:
- Regularly review and update content
- Remove duplicate items
- Keep active items count manageable
- Archive old seasonal content

## üîß Troubleshooting

### Item not showing on website?
- Check if status is set to **Active**
- Verify the URL is accessible
- Clear browser cache

### Image not loading?
- Verify URL is correct and publicly accessible
- Check if URL uses HTTPS
- Try opening URL in new browser tab

### Can't edit item?
- Ensure you're logged in as admin
- Refresh the page
- Check browser console for errors

### Filters not working?
- Clear all filters and try again
- Refresh the page
- Check if items match filter criteria
